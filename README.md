# Lab 6 Python Repositories

This workspace contains two separate exercise repositories for Lab 6:

- `exercise1_graph_representations`
- `exercise2_graph_traversal_dfs`

Each folder contains:

- simple Python code
- a small runnable example

Run each exercise from inside its own folder with:

```bash
python3 main.py
```

## Exercises

- Exercise 1: graph representations with adjacency matrix and adjacency lists
- Exercise 2: graph traversal with DFS and connected components
